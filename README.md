## 📱 프로젝트 소개
React Native를 처음 배우면서 만든 앱입니다.  
이 앱은 사용자가 **위험한 상황에서 안전을 지키기 위한 기능**을 제공합니다.  
앱의 **UI 디자인과 기능 기획**은 모두 제가 직접 진행했습니다.  
무단 아이디어 도용을 금지합니다.

## 🛠️ 사용 기술
- React Native 0.82.1
- TypeScript
- iOS Simulator
- Styled Components (스타일링)
- React Navigation (화면 전환)

## 🖼️ 주요 화면
1. **가이드 화면**: 앱 소개 및 기능 안내  
2. **프로필 화면**: 사용자 정보 입력  
3. **안전 기능 화면**: 긴급 상황 알림 및 위치 공유
